# 📸 사진(스크린샷) 넣는 방법

블로그 글마다 **점선 박스(사진 슬롯)** 가 있어요. 여기에 실제 게임 스크린샷을 넣으면 됩니다.
전부 **가만히 서서 한 장이면 되는, 찍기 쉬운 장면** 위주로 골랐어요.

## 1) 어떤 사진을 찍으면 되나요? (추천 목록)

| 파일 이름(권장) | 어떤 화면 | 어디에 쓰여요 |
|---|---|---|
| `title.jpg` | 게임 타이틀/로그인 첫 화면 | 첫날 후기 |
| `create.jpg` | 캐릭터 생성 화면 | 첫날 후기 |
| `village.jpg` | 첫 마을 광장 + 내 캐릭터 | 첫날 후기 |
| `select.jpg` | 캐릭터 선택 화면 | 캐릭터 가이드 |
| `myfox.jpg` | 내 캐릭터 정보창(스탯+이름) | 캐릭터 가이드 |
| `drill-gauge.jpg` | 드릴 게이지 뜬 삽질 화면 | 드릴 공략 |
| `drill-loot.jpg` | 발굴 성공/아이템 획득 팝업 | 드릴 공략 |
| `inventory.jpg` | 인벤토리/소지금 화면 | 무과금 꿀팁 |
| `growth.jpg` | 레벨업 순간 or 강해진 캐릭터 | 무과금 꿀팁 |
| `nick-proof.jpg` | **닉네임 "호떡해"가 보이는 정보창** (제일 중요!) | 이벤트 인증 |
| `apply.jpg` | 보상 신청 화면/게시글 | 이벤트 인증 |

> 💡 게임 스크린샷은 보통 키보드 **`Print Screen`** 키 → 그림판 붙여넣기 → 저장,
> 또는 Windows **`Win + Shift + S`** (캡처 도구)로 찍으면 됩니다.

## 2) 찍은 사진을 이 `images` 폴더에 넣기

위 표의 파일 이름으로 저장해서 이 폴더(`images/`)에 넣어주세요.

## 3) 글에서 점선 박스를 실제 사진으로 바꾸기

각 글(HTML) 파일을 열면 사진 슬롯마다 이런 주석이 있어요:

```html
<!-- 실제 사진으로 교체: <img src="../images/title.jpg" alt="포푸리트릭스터 타이틀 화면"> -->
<div class="photo-slot">...📸 안내문...</div>
```

이 `<div class="photo-slot">...</div>` 줄을 지우고, 주석 안의 `<img ...>` 한 줄로 바꾸면 끝!

**예시 (바꾸기 전 → 후):**
```html
<!-- 전 -->
<div class="photo-slot"><span class="cam">📸</span>찍기 쉬운 사진 ①<br>게임 <b>타이틀 / 로그인 화면</b> 캡처</div>

<!-- 후 -->
<img src="../images/title.jpg" alt="포푸리트릭스터 타이틀 화면">
```

사진을 아직 안 넣어도 점선 박스가 예쁘게 표시되니, **글 먼저 올리고 사진은 나중에 채워도** 괜찮아요.
