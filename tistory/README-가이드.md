# 🌸 티스토리 이전 & 애드센스 수익화 가이드

이 폴더(`tistory/`)에는 지금 블로그 글들을 **티스토리에 그대로 붙여넣을 수 있게 변환한 HTML**이 들어 있어요.
각 파일 맨 위 주석에 **제목**이 적혀 있고, 본문은 **인라인 스타일**이라 티스토리 기본 에디터에서도 디자인이 유지됩니다.

## 📁 파일 목록
- `01-first-day.html` ~ `08-card-identify.html` … 글 본문 (붙여넣기용)
- `00-소개.html` … 블로그 소개 페이지 (애드센스 승인에 유리)
- `00-개인정보처리방침.html` … **애드센스 사실상 필수 페이지**
- (이벤트글 05번은 깃허브 블로그 전용이라 제외했어요)

---

## 1단계. 티스토리 블로그 만들기
1. <https://www.tistory.com> → 카카오 계정으로 로그인 → **블로그 개설**
2. 주소(예: `hotteokhae.tistory.com`)와 블로그 이름 설정
3. **스킨**은 반응형(모바일 대응) 스킨 추천 (예: "Odyssey", "Poster" 등 기본 스킨)

## 2단계. 글 발행하기 (붙여넣기)
1. **글쓰기** → 오른쪽 위 에디터 모드를 **`기본모드` → `HTML`** 로 변경
2. 이 폴더의 `.html` 파일을 **메모장으로 열어** 맨 위 주석 아래 내용을 **전체 복사**
3. HTML 편집창에 **붙여넣기**
4. 다시 **`기본모드`로 전환**하면 미리보기 확인 가능
5. **제목칸**에는 파일 맨 위 주석에 적힌 제목을 넣기
6. 📸 `여기에 스크린샷 삽입` 박스는 → 실제 게임 스크린샷을 그 자리에 **이미지 업로드**로 교체 (박스 문단은 지우기)
7. **카테고리**를 만들어 분류 (예: `플레이후기` / `초보가이드` / `공략`)
8. 발행!

> 💡 팁: 글마다 **직접 찍은 스크린샷 1~3장**을 꼭 넣으세요. 오리지널 이미지가 있으면 애드센스 승인·검색 노출 모두 유리해요.

## 3단계. 필수 페이지 만들기 (애드센스 대비)
1. `00-개인정보처리방침.html` → 발행 후 **블로그 메뉴/사이드바에 링크** 걸기
   - ⚠️ 파일 안 `여기에_본인_이메일_입력@example.com` 을 **본인 이메일로 꼭 수정**
2. `00-소개.html` → 소개 페이지로 발행 후 메뉴에 걸기
3. (권장) 관리 → **꾸미기/메뉴**에서 상단 메뉴에 `소개`, `개인정보처리방침` 노출

## 4단계. 구글 애드센스 신청
1. <https://adsense.google.com> → 구글 계정으로 시작
2. 사이트에 **티스토리 주소**(`xxx.tistory.com`) 입력
3. 안내에 따라 **애드센스 연동 코드**를 받으면 →
   - 티스토리 **관리 → 수익 → 애드센스** 메뉴에서 연결하거나,
   - 스킨 편집의 `<head>` 안에 코드 삽입
4. `ads.txt` 요청이 뜨면 티스토리 애드센스 연동 기능이 자동 처리해줘요 (수동이면 안내대로)
5. **심사 대기** (보통 며칠~2주). 승인되면 광고가 자동 노출됩니다.

## 5단계. 승인 확률 높이는 체크리스트
- ✅ **글 15~20개 이상** (지금 7개 → 더 채우면 좋아요. 제가 계속 도와드릴 수 있어요!)
- ✅ 글마다 **충분한 분량 + 직접 찍은 이미지**
- ✅ **개인정보처리방침 + 소개** 페이지 존재
- ✅ 메뉴/카테고리로 **탐색이 명확**
- ✅ 저작권 주의: 게임 스크린샷은 **직접 촬영분** 위주, 남의 이미지·펌글 금지
- ✅ 꾸준한 발행 (심사 중에도 글을 계속 올리면 유리)

## ⚠️ 주의: 티스토리 광고 정책
티스토리(카카오)는 **자체 광고를 함께 노출**하고 정책이 종종 바뀝니다.
가입 후 **관리 → 수익** 메뉴에서 최신 정책과 애드센스 연동 방식을 꼭 확인하세요.
