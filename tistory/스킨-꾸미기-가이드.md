# 🎨 티스토리 스킨 & 가독성 꾸미기 가이드

블로그를 **읽기 편하고 예쁘게** 만드는 방법이에요.
새 스킨 파일을 통째로 올리는 대신, **검증된 기본 스킨 + 가독성 CSS 덧씌우기**로 갑니다.
→ 화면이 깨질 위험이 없고, 나중에 색/폰트도 쉽게 바꿀 수 있어요.

---

## 1단계. 어떤 기본 스킨을 고를까?

**관리 → 꾸미기 → 스킨** 에서 아래 중 하나를 적용하세요. (셋 다 반응형·애드센스 OK)

| 스킨 | 특징 | 추천 대상 |
|------|------|-----------|
| **Odyssey** ⭐ | 깔끔·군더더기 없음, 티스토리 대표 스킨 | **가장 무난. 이걸로 시작하세요** |
| **Book Club** | 읽기(독서)에 특화, 여백 넉넉 | 글 위주로 보여주고 싶을 때 |
| **Poster** | 큰 썸네일 카드형, 감성적 | 스크린샷을 크게 자랑하고 싶을 때 |

> 💡 우리 블로그는 **글 본문이 이미 예쁘게 꾸며져(인라인 스타일)** 있어서, 스킨은 "테두리·메뉴·배경"만 담당해요. 그래서 어떤 스킨을 골라도 본문 디자인은 그대로 유지됩니다.

---

## 2단계. 가독성 CSS 붙여넣기 (핵심!)

1. **관리 → 꾸미기 → 스킨편집** 클릭
2. 오른쪽 위 **`html 편집`** 버튼 클릭
3. 상단 탭에서 **`CSS`** 탭 선택
4. **CSS 내용 맨 아래 끝으로 스크롤** → 같은 폴더의 **`custom-skin.css`** 파일 내용을 **전체 복사해서 붙여넣기**
5. 오른쪽 위 **`적용`** 클릭 → 끝!

이걸 넣으면 이렇게 바뀌어요:
- ✅ 한글 전용 폰트(프리텐다드)로 또렷하게
- ✅ 줄간격·글자 크기 넉넉하게 (모바일 포함)
- ✅ 한글 단어가 줄 끝에서 안 잘리게 (`word-break`)
- ✅ 본문 폭을 눈이 편한 길이로 (너무 넓게 안 퍼짐)
- ✅ 이미지 모서리 둥글게 + 그림자, 표는 모바일에서 가로 스크롤
- ✅ 브랜드 색(주황·크림·형광펜 노랑)으로 통일

---

## 3단계. 색/폰트 바꾸고 싶으면?

`custom-skin.css` 맨 위 **`:root { ... }`** 부분의 색 코드만 바꾸면 전체에 반영돼요.
예) 포인트색을 바꾸려면 `--accent: #c85a3d;` 값만 원하는 색으로 교체.

## 4단계. 블로그 기본 정보 세팅 (꾸미기 완성도 ↑)

**관리 → 블로그 / 꾸미기 메뉴**에서:
- **블로그 제목**: `🌸 호떡해의 포푸리트릭스터 블로그`
- **블로그 설명**: `추억의 2D게임 트릭스터, 포푸리 서버 플레이 일기 · 초보 가이드`
- **상단 메뉴**에 카테고리 노출: `플레이후기` / `초보가이드` / `공략` / `소개` / `개인정보처리방침`
- **대표 이미지(파비콘·썸네일)**: 게임 스크린샷이나 캐릭터 아이콘 1장 등록하면 SNS 공유 시 예쁘게 나와요

---

## 5단계. (나중에) 애드센스 코드는 어디에?

승인 후 광고 코드는 **스킨편집 → html 편집 → `HTML` 탭 → `<head>` 안**에 붙여넣거나,
**관리 → 수익 → 애드센스** 메뉴에서 자동 연동하면 돼요.
(자세한 건 `README-가이드.md` 4단계 참고)

---

### ⚠️ 폰트가 안 바뀌는 것 같다면
프리텐다드 폰트는 외부에서 불러오는데, 아주 드물게 차단될 수 있어요.
그래도 걱정 마세요 — CSS에 **애플/노토 한글 폰트로 자동 대체**되도록 해놨으니 가독성은 그대로 유지됩니다.
