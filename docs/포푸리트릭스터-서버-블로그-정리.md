## 3. 커뮤니티 총평 (장단점 종합)

### 👍 장점
- **2주마다 대규모 패치** — 신규 콘텐츠·이벤트가 자주 열림
- **높은 배율**(경험치 3배 / TM 6배)로 육성 부담 ↓
- **강화된 마을장비 + 에피소드 보상**으로 무과금 정착이 쉬움
- **공식 디스코드 소통 활발** (1~2일 내 답변)
- 캐시 아이템 기간 무제한 → 무과금도 꾸밀 거리 확보
- 폴라 대비 **반응속도가 빠름**

### 👎 단점
- **신규 서버라 유저수가 적음** → 대부분 자급자족 필요
- **드랍률이 후하지 않음** (넷마블 시절과 비슷하거나 약간 높은 수준)
- **한글화 부족** (마이샵 등 일부 영어 표기)
- 마이샵 환전율이 폴라(3,000pt)보다 높음(5,000pt : 1개)
- 밸런스: 일부 **매직 캐릭터가 과하게 강하다**는 의견

## 4. 포푸리 vs 폴라 서버 비교

| 항목 | 🌸 포푸리(ㅍㅍㄹ) | ❄️ 폴라(ㅍㄹ) |
|---|---|---|
| 운영 기간 | 신규 (2024-12 오픈) | 약 5년, 안정적 |
| 유저수 | 적음 | 많음 (템 구하기 쉬움) |
| 업데이트 | **2주마다 대규모 패치** | 1~2개월에 스킨 추가 정도 |
| 운영 소통 | 공식 디스코드, 1~2일 | 디스코드 없음, 1~2개월 |
| 배율 | 경험치 3배 / TM 6배 | — |
| 마이샵 환전 | 5,000pt : 1개 | 3,000pt : 1개 |
| 드랍률 | 보통 | 상대적으로 나음 |
| 추천 대상 | **무과금·신규·소통 중시** | **과금·안정성·유저풀 중시** |

> **한 줄 결론**: 과금·안정성이면 폴라, **무과금으로 활발한 업데이트를 즐기려면 포푸리**.
